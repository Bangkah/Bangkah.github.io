1. Web 1.0 - Era Web Statis

Web 1.0 merupakan generasi awal dari teknologi web yang muncul pada tahun 1990-an. Web pada masa ini bersifat statis, artinya kontennya tidak dapat berubah secara dinamis tanpa modifikasi langsung pada kode HTML-nya.

  • Konten hanya dapat dibaca (read-only)
  • Tidak ada interaksi dari pengguna
  • Website bersifat personal atau informasi perusahaan
  • Teknologi utama: HTML, gambar, hyperlink

2. Web 2.0 - Era Interaktif dan Dinamis

Web 2.0 ditandai dengan munculnya aplikasi web yang memungkinkan pengguna berkontribusi secara aktif, seperti media sosial, blog, dan forum.

  • Konten dapat dibuat dan diubah oleh pengguna
  • Interaksi real-time menggunakan AJAX
  • Perpindahan dari halaman statis ke halaman dinamis
  • Penggunaan CMS (Content Management System) seperti WordPress dan Joomla
  • Teknologi: HTML, CSS, JavaScript, PHP, MySQL, AJAX

3. Web 3.0 - Web Semantik dan Cerdas

Web 3.0 dikenal juga sebagai Semantic Web yang berfokus pada pemrosesan data secara lebih cerdas. Web mulai menggunakan AI dan machine learning untuk personalisasi konten.

  • Pemanfaatan data terstruktur (JSON-LD, RDF)
  • Interkoneksi data antar aplikasi
  • Konten disesuaikan dengan pengguna (personalized content)
  • Penggunaan teknologi seperti AI, blockchain, dan Big Data

4. Web Programming Modern

Teknologi pengembangan web modern menggabungkan frontend dan backend dengan pendekatan modular dan efisien.

Frontend (Client-Side)

  • Framework: React, Angular, Vue.js, Svelte
  • Bahasa: HTML5, CSS3, JavaScript (ES6+)
  • Build tools: Webpack, Vite, Babel

Backend (Server-Side)

  • Bahasa: Node.js, Python, PHP, Ruby, Java
  • Database: MySQL, PostgreSQL, MongoDB, Firebase
  • Server: Apache, Nginx, Express.js

5. Single Page Application (SPA)

SPA adalah jenis aplikasi web yang hanya memuat satu file HTML utama dan memperbarui konten secara dinamis menggunakan JavaScript, tanpa reload
